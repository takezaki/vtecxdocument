# @vtecx/vtecxdocument

vte.cx BaaS framework and API documentation package.

## Contents

- `docs/framework.md` — Framework overview
- `docs/vtecxnext-api.md` — vte.cx Next API guide
- `docs/api/` — Individual API reference documents

## Usage

```bash
npm install @vtecx/vtecxdocument
```

After installation, documentation is available under `node_modules/@vtecx/vtecxdocument/docs/`.
